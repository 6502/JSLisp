How to specify key bindings

Each key binding requires a line with the key combination and a command separated by spaces.

Key combination syntax

The key combination can be preceded by zero or more prefixes:

  • ctrl-
  • shift-
  • alt-

Note that in case of multiple prefixes the order must be respected.

After prefix part (that may be empty) there is the key name. A few special keys have a specific name: Left, Right, Up, Down PgUp, PgDn, Home, End, Space, Backspace, Del, Tab, Esc, Enter, F1, F2, F3, ..., F12.

In addition to those special keys any uppercase character or digit can be used to represent the correspondig key.

To create a mapping that involves a key that is not listed otherwise it is also possible to use for example #197 to specify the key for which event.which has value 197 in the processing of onkeydown Javascript event.

For example are valid key combinations:

  • ctrl-Left
  • alt-W
  • ctrl-alt-2
  • F11
  • alt-#197

While instead are invalid:

  • shift-ctrl-Left (modifiers order must be preserved)
  • ctrl-$ (only A-Z and 0-9 can be used)

Command syntax

The command associated to a key combination is a single word (optionally containing hypens). Which commands are acceptable depends on the section (IDE or EDITOR).
