Compile specialization: and
(and x1 x2 ... xn)
Returns the value of last form 'xn' if all forms evaluate to logically true or otherwise returns the first logically false result without evaluating subsequent forms.
Compile specialization: append
(append list-1 list-2 ... list-n)

Compile specialization: apply
No documentation available
Compile specialization: cond
(cond ((t1 f1)(t2 f2)...(tn fn)))
Evaluates in sequence t1, t2 ... tn and returns the value of the first corresponding form 'f' when the value is logically true without evaluating subsequent conditions. Returns null if no condition t evaluates to logically true
Compile specialization: defvar
(defvar variable:symbol value)
Sets the value cell of variable only if is not currenly undefined, and also marks the symbol as 'special' so that future value bindings on this symbol will always be dynamic and not lexical.
Compile specialization: do
(do ((v1 init1 [inc1])...)(exit-test res1 res2 ...) b1 b2 ...)
Loops over the body forms b1 b2 ... bn by first establishing a lexical/dynamic binding v1=init1, v2=init2, ... and by assigning the value of the increment forms inc1 to v1, inc2 to v2 ... where they are present after each iteration. Before entering each loop iteration the exit-test form is evaluated and if logically true the iteration is not performed and the result forms res1, res2 ... are evaluated in sequence with the value of last of them being used as the final result of the (do ...) form.
Compile specialization: dolist
(dolist (var x:list) f1 f2 ... fn)
Evaluates the body forms f1 f2 ... fn in sequence times by setting the dynamically/lexically bound variable 'var' to next element of list 'x' each time. The return value is null.
Compile specialization: dotimes
(dotimes (var count) f1 f2 ... fn)
Evaluates the body forms f1 f2 ... fn in sequence exactly 'count' times by setting the dynamically/lexically bound variable 'var' to 0, 1, ... count-1 before each iteration. The return value is null.
Compile specialization: if
(if condition then-part [else-part])
Conditional evaluation form. Evaluates either then-part only or else-part only depending on wether the evaluation of condition returned a true value or not.
Compile specialization: js-code
(js-code x:string-literal)
Verbatim javascript code generation.
Compile specialization: labels
(labels ((func1 (x1 x2 ... xn) f1 f2 .. fn)...) b1 b2 ... bn)
Excutes the body forms b1, b2 ... bn by first establishing a lexical binding for the function names func1, func2 ... funcn. When compiling the body forms any macros defined outside the (labels ...) form with names func1, func2, ... funcn will be ignored.
Compile specialization: lambda
(lambda (arg-1 ... arg-n) form-1 form-2 ... form-n)
Returns a function object that when called will lexically/dynamically bind parameters to arg1, arg2, ... arg-n and that will evaluate form-1 form-2 form-n in sequence returning the last evaluated form value as result
Compile specialization: let
(let ((x1 v1)(x2 v2) ... (xn vn)) f1 f2 ... fn)
Evaluates a sequence of forms f1, f2 ... fn by first establishing lexical/dynamic bindings for the variables x1=v1, x2=v2 ... xn=vn. The evaluation of the forms v1 ... vn does /NOT/ consider the bindings that will be established by (let ...).
Compile specialization: macrolet
(macrolet ((m1 (x1 x2 ...) b1 b2 ...) ...) body1 body2 ... bodyn)
Evaluates the body forms body1, body2, ... bodyn that are compiled by first installing the lexical macros m1, m2 ... mn. Global macros accessible with (symbol-macro x) are not affected by these local definitions.
Compile specialization: not
(not x) -> bool
Returns true if and only if x is logically false and false otherwise.
Compile specialization: or
(or x1 x2 ... xn)
Returns the value of the first form that evaluates to logically true without evaluating subsequent forms, or otherwise returns the value of last form 'xn' if all of them evaluate to logicall false.
Compile specialization: progn
(progn form-1 form-2 ... form-n)
Evaluates all the forms in sequence, returning as value the value of the last one.
Compile specialization: quote
(quote x)
Returns the unevaluated x as result.
Compile specialization: setq
(setq name value)
Sets the current value of variable 'name'. When 'name' is currently bound to a symbol macro setq is transformed in a corresponding (setf ...) form.
Compile specialization: symbol-macrolet
(symbol-macrolet ((x1 def1)(x2 def2)...) body1 body2 ... bodyn)
Evaluates the body forms body1, body2 ... bodyn that are compiled by first installing the lexical symbol macros x1 x2 ... xn.
Compile specialization: try
(try expr on-error)
Evaluates expr and in case of exception evaluates the on-error form setting *exception* to the current exception
Compile specialization: unless
(unless condition f1 f2 ... fn)
If 'condition' evaluates to logically false evaluates the body forms f1, f2, ... fn in sequence an returns the value of last evaluated form, otherwise returns null without evaluating any of the body forms.
Compile specialization: when
(when condition f1 f2 ... fn)
If 'condition' evaluates to logically true evaluates the body forms f1, f2, ... fn in sequence an returns the value of last evaluated form, otherwise returns null without evaluating any of the body forms.
